# IPv6 Test for Command line

## Usage

```console
$ ./go-ipv6-test -h
Usage of ./go-ipv6-test:
  -all
    	Test All Sites
  -host string
    	Host (default "test-ipv6.com")
  -sites
    	Available Sites
```

## Build

```bash
goreleaser build
```

more see [GoReleaser](https://goreleaser.com)
