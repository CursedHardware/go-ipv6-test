# IPv6 Test for Command line

## Usage

```console
$ ./go-ipv6-test -h
Usage of ./go-ipv6-test:
  -host string
    	Host (default "main.test-ipv6.com")
  -mirrors
    	List of all mirror sites
```

## Build

```bash
goreleaser build
```

more see [GoReleaser](https://goreleaser.com)
