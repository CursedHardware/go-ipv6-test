# IPv6 Test for Command line

## Usage

```console
$ ./go-ipv6-test -h
Usage of ./go-ipv6-test:
  -all
    	Test All Hosts
  -host string
    	Host (default "jp2.test-ipv6.com")
  -hosts
    	Available Hosts
```

## Build

```bash
goreleaser build
```

more see [GoReleaser](https://goreleaser.com)
